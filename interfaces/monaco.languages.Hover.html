Interface Hover

Source
Expand description

A hover represents additional information for a symbol or word. Hovers are rendered in a tooltip-like widget.

interface Hover {
    canDecreaseVerbosity?: boolean;
    canIncreaseVerbosity?: boolean;
    contents: IMarkdownString[];
    range?: IRange;
}
